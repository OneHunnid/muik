Buttons

Buttons come in many flavors; colors, rounded corners, shadow, border and borderInverse. To style a button, simply add the button modifiers to an anchor tag or button element.

button button--dark button--medium button--light button--rounded button--shadow button--border
Inputs

Inputs have an assortment of modifiers; colors, rounded corners, shadow and border. To style an input, simply add the input modifiers.
